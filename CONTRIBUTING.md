# Contributing to Git World

Thank you for your interest in contributing!

## How to Contribute

1. **Fork the repository**
2. **Create a new branch** for your feature or bugfix
3. **Make your changes**
4. **Test thoroughly**
5. **Submit a pull request**

## Guidelines
- Use clear, descriptive commit messages
- Follow the project structure and naming conventions
- Write clean, readable code (TypeScript, React best practices)
- Add comments for complex logic
- Update documentation if needed

## Reporting Issues
- Use the issue template for bug reports or feature requests
- Provide steps to reproduce, screenshots, and expected behavior

## Code Style
- TypeScript, React 19, Next.js 16
- Tailwind CSS for styling
- Use Zustand for state management
